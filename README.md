# Description
Libft is the first project at the 42 school as a cadet. The objective is to rewrite in your own way some libc functions, that will be used in future projects.

# Books on the library
At the moment, the following functions are at disposal:
- **ft_atoi**      	   	     // ASCII to integer.
- **ft_bzero**		     // Erases n bytes starting at where s points to and populate with zeroes.
- **ft_calloc**		     // Allocate memory and initialize it with zeroes.
- **ft_isalnum**		     // Verify if is alphanumeric.
- **ft_isalpha**		     // Verify if is alphabetical.
- **ft_isascii**		     // Verify if is ASCII.
- **ft_isdigit**		     // Verify if is a digit.
- **ft_isprint**		     // Verify if is a printable character.
- **ft_memchr**		     // Scan n bytes in memory in search of a character c
- **ft_memcmp**		     // Compare memory areas.
- **ft_memcpy**		     // Copy n bytes from source to destiny. 
- **ft_memmove**		     // Same as memcpy but with overlap in memory.
- **ft_memset**		     // Fill the first n bytes of memory with a constant byte.
- **ft_strchr**		     // Return the pointer to the first ocurrence of character n in a given string.
- **ft_strdup**		     // Duplicate a string.
- **ft_strlcat**		     // String concatenation in a safer way, due to the full size of the buffer.
- **ft_strlcpy**		     // Same as strlcat, but copying instead of concatenate.
- **ft_strlen**		     // Return the size of a string.
- **ft_strncmp**		     // Compare two strings, n bytes at most.
- **ft_strnstr**		     // Find a substring in a given string with upper bound size.
- **ft_strrchr**		     // Return a pointer to the last ocurrence of a given character in a given string.
- **ft_tolower**		     // Lowercase the given character.
- **ft_toupper**		     // Uppercase the given character.
- **ft_putnbr_fd**		     // Output integer to the file descriptor.
- **ft_putendl_fd**		     // Output string with new line to the file descriptor.
- **ft_putstr_fd**		     // Output string to the file descriptor. 
- **ft_putchar_fd**		     // Output character to the file descriptor.
- **ft_striteri**	     // Apply a function f to every character of a given string. 
- **ft_strmapi**		     // Same as striteri but create a new string.
- **ft_itoa**		     // Integer to ASCII.
- **ft_split**		     // Split a string given a delimiter character and return an array of strings.
- **ft_strtrim**		     // Trim the begin and end of a string with specified characters.
- **ft_strjoin**		     // Concatenate two strings
- **ft_substr**		     // Find a substring in a string with delimited size.
- **ft_lstnew**		     // Create a linked list.
- **ft_lstadd_front**	     // Add element to the begin of a linked list.
- **ft_lstsize**		     // Return the size of a linked list.
- **ft_lstlast**		     // Return the last element of a linked list.
- **ft_lstadd_back**	     // Add element to the end of a linked list.
- **ft_lstdelone**		     // Free the memory of a given element from a linked list. 
- **ft_lstclear**		     // Delete and clear all elements of a linked list after a given one.
- **ft_lstiter**		     // Iterate every element of a linked list with a function f.
- **ft_lstmap**		     // Iterate every element of a linked list with a function f and create a new list with those elements.
